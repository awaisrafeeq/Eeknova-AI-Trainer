12. COMMAND LINE INTERFACE

----------------------------------------


12.1 USAGE EXAMPLES
:
------------------------------
bash

============================================================
                    WITH FEEDBACK MODE
                     
============================================================
python main.py --choice 1 --video_path "camera" --target_move "salsa_step"


============================================================
                    SKELETON ONLY MODE
                     
============================================================
python main.py --choice 2 --video_path "video.mp4" --output_path "output.mp4"


============================================================
                   VIDEO FILE PROCESSING
                   
============================================================
python main.py --choice 1 --video_path "dance_video.mp4" --target_move "twist_step"



12.2 PARAMETERS
:
------------------------------
- --choice: 1 (feedback), 2 (skeleton only), 3 (exit)
- --video_path: Video file or "camera" for live mode
- --target_move: Dance move name (required for feedback mode)
- --output_path: Output video file (optional)
- --include_audio: Include audio in output (default: True)
